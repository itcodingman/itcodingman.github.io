Spring Security 注册教程

为您的 Web 应用程序构建一个成熟的、可用于生产的注册不仅仅是将一个简单的注册页面放在一起。

有 很多问题 需要回答：

  • 如何 验证新用户的电子邮件地址 ？
  • 如何正确安全地 存储用户凭据？
  • 如果用户 忘记了密码怎么办？
  • 用户更改自己的密码怎么 办？
  • 密码应该有多 强？如何在应用程序中强制执行一些合理的默认设置，以便我的用户拥有良好、强密码？
  • 如果我有不止一种类型的用户怎么办？我需要一种 存储角色和权限的好方法。
  • 安全问题呢 ？我什至应该拥有它们吗？
  • 如何在良好的本地化支持下完成所有这些工作？涉及的消息很多。
